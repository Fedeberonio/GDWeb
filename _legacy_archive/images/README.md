# Imágenes Archivadas

Este directorio contiene imágenes que han sido identificadas como duplicadas o no utilizadas.

## Contenido

- **Imágenes duplicadas**: Imágenes que fueron identificadas como duplicados explícitos de `legacy/image_duplicates/`
- **Imágenes no utilizadas**: Imágenes de `legacy/image_originals/` que no están en uso activo en `apps/web/public/`

## Notas

- Estas imágenes se mantienen aquí para referencia histórica
- Las imágenes activas se encuentran en `apps/web/public/assets/images/`
- Si necesitas alguna imagen, verifica primero en la ubicación activa

## Estructura Original

- `legacy/image_originals/` - 108 imágenes originales (mantenidas para referencia)
- `legacy/image_duplicates/` - 53 imágenes duplicadas (movidas aquí)
- `apps/web/public/assets/images/` - 161 imágenes activas en uso
